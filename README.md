# 地理探索助手 🌍

一个专为图寻（类似GeoGuessr）游戏设计的智能浏览器扩展，通过AI视觉分析帮助识别地理位置。

## ✨ 功能特色

- 🔍 **智能位置分析**: 使用AI视觉模型分析截图中的地理特征
- 🎯 **精确定位**: 分析地貌、建筑、路标、文字等多维度信息
- ⚡ **一键操作**: 简单点击即可开始分析
- 🛠️ **自定义配置**: 支持自定义API域名、密钥和模型
- 🌐 **多域名支持**: 兼容tuxun.fun和Google Maps等网站
- 📱 **现代界面**: 简洁美观的用户界面

## 🎮 支持的游戏网站

- [图寻 (tuxun.fun)](https://tuxun.fun) - 主要支持
- Google Maps 相关页面
- 其他地理猜测类游戏

## 📦 安装方法

### 开发者模式安装

1. 下载或克隆此项目到本地
2. 打开Chrome浏览器，进入扩展管理页面 (`chrome://extensions/`)
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 扩展安装完成！

## ⚙️ 配置设置

首次使用需要配置API设置：

1. 点击浏览器工具栏中的扩展图标
2. 填写以下信息：
   - **API 域名**: 你的AI服务API地址（如：`https://api.openai.com`）
   - **API Key**: 你的API密钥
   - **模型**: AI模型名称（默认：`gemini-2.0-flash`）
3. 点击"保存设置"

## 🚀 使用方法

1. 打开支持的游戏网站（如 tuxun.fun）
2. 进入游戏界面
3. 点击扩展图标，然后点击"开始分析位置"
4. 等待AI分析完成
5. 查看分析结果，格式为：`大洲 > 国家(位于大洲的方位) > 市(位于国家的方位)`

### 示例输出

```
非洲 > 马达加斯加(非洲的东南部) > 贝富塔卡(马达加斯加的南部)
```


### 核心功能

- **截图捕获**: 自动获取当前页面截图
- **AI分析**: 调用视觉AI模型分析地理特征
- **结果展示**: 以toast形式显示分析结果
- **设置管理**: 本地存储API配置信息

## 🛡️ 权限说明

扩展需要以下权限：

- `activeTab`: 访问当前活动标签页
- `storage`: 存储用户设置
- `scripting`: 注入脚本和样式
- `tabs`: 管理标签页
- `desktopCapture`: 捕获屏幕截图

## 🔒 隐私保护

- 所有设置信息仅存储在本地浏览器中
- 截图数据仅用于AI分析，不会被永久存储
- 支持自定义API服务，数据处理完全由用户控制

## 🐛 故障排除

### 常见问题

1. **分析失败**
   - 检查API域名和密钥是否正确
   - 确认网络连接正常
   - 验证API服务是否可用

2. **无法截图**
   - 确认在支持的网站上使用
   - 检查浏览器权限设置

3. **结果不准确**
   - 尝试更换AI模型
   - 确保截图内容清晰可见
